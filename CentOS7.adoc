=== CentOS 7 常用命令

==== 目录

. mkdir
+
创建目录

. cd
+
进入目录

. rmdir
+
删除目录

. rm
+
删除文件或目录。
+
.. [red]*rm -rf*
强制删除目录下所有的东西，-r 表示递归，-f 表示强制。

==== 文件

. touch
+
创建文件或修改文件时间

. cp
+
拷贝文件
+
----
cp myFile myFile.bk
----

. mv
+
移动文件
+
----
mv myFile myDir
----

==== 其他

. clear
+
清屏

. man
+
显示用法手册
+
----
# 如果还未安装手册，可用如下命令
yum install man-pages
----
