== 常用

=== 整数

----
^-?d+$
----

=== 正整数

----
^[1-9]+d*$
----

=== 负整数

----
^-{1}d+$
----

=== 非负整数

----
^d+$
----

=== 非负浮点数

----
^[0-9]*[.][0-9]+$
----

=== 正浮点数

----
^0*[1-9][0-9]*(\.[0-9]+)?|0+\.[0-9]*[1-9][0-9]*$
----

=== 英文字母

----
^[a-zA-Z]+$
----

=== 英文数字

----
^[a-zA-Z0-9]+$
----

=== Email

----
^w+([-+.]w+)*@w+([-.]w+)*.w+([-.]w+)*$
----

=== IP地址

----
^((2[0-4]d|25[0-5]|[01]?dd?).){3}(2[0-4]d|25[0-5]|[01]?dd?)$
----

=== 汉字

----
^[u4E00-u9FA5]+$
----
